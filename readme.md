# Spotify Tracker

## Instructions for Build and Use

Steps to build and/or run the software:

1. Clone the repository from GitHub
2. Install python, django, and spotipy
3. Cd into the main directory containing manage.py
4. Run the command `python manage.py runserver`

Instructions for using the software:

1. Once the server is running, follow the link to the login page
2. Login with your Spotify credentials and authorize the app
3. Use the web interface to explore your Spotify data

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

- Download python, django, and spotipy
